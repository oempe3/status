# Painel de Disponibilidade - Integração Google Sheets

1. Faça login como "admin" (admin/admin) para editar, ou "operador" para apenas visualizar.
2. status.html mostra os dados em tempo real do Google Sheets.
3. entrada.html permite atualizar individualmente o status dos equipamentos.
4. csv_update.html faz atualização em lote via CSV (pode ser copiado do Excel).
5. Todos os arquivos já usam seu endpoint do Apps Script.

Qualquer dúvida ou melhoria, peça para o ChatGPT adaptar!
